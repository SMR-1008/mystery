# C语言简易进制转换工具 (Base Converter)

这是一个基于 C 语言编写的轻量级命令行交互程序。它利用了 C 语言标准库中的格式化输入输出特性，实现了十进制（Decimal）、八进制（Octal）和十六进制（Hexadecimal）之间的快捷相互转换。

## ✨ 功能特点

该程序支持以下 6 种进制转换模式，并包含一个退出指令：

1. **10 to 8**: 十进制转换为八进制
2. **10 to 16**: 十进制转换为十六进制
3. **8 to 16**: 八进制转换为十六进制
4. **16 to 8**: 十六进制转换为八进制
5. **8 to 10**: 八进制转换为十进制
6. **16 to 10**: 十六进制转换为十进制
7. **退出程序**

## 🚀 编译与运行

请确保你的计算机上已经安装了 C 语言编译器（如 GCC）。

**1. 编译代码**
打开终端或命令行，进入代码所在的目录，运行以下命令将代码编译为可执行文件（假设你的文件名为 `converter.c`）：

```bash
gcc converter.c -o converter

```

**2. 运行程序**
编译成功后，在终端中运行生成的可执行文件：

* **Windows:**
```cmd
converter.exe

```


* **Linux / macOS:**
```bash
./converter

```



## 💡 使用说明

1. 运行程序后，屏幕会打印出功能菜单。
2. 在 `请输入序号:` 提示符后，输入对应的**数字序号 (0-6)** 并按下回车，选择你需要的转换模式。
3. 接着输入你需要转换的**数值**并按下回车。
* *注意：输入八进制数时不需要加前缀 `0`，输入十六进制数时不需要加前缀 `0x`，程序会自动根据你选择的模式识别。*


4. 程序会立即输出转换后的结果。
5. 程序将持续循环运行，直到你输入序号 `0` 退出。

## 🛠️ 技术原理

本程序的核心逻辑非常精简，主要依赖于 `<stdio.h>` 中的 `scanf` 和 `printf` 格式化占位符进行自动转换：

* `%d`：用于读取和打印十进制整数。
* `%o`：用于读取和打印无符号八进制整数。
* `%x`：用于读取和打印无符号十六进制整数（小写）。

程序在接收用户输入时直接将其格式化为对应进制的内存数据，并在输出时再次格式化为目标进制，避免了复杂的数学计算过程。
