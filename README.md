# C语言进阶进制转换工具 (Advanced Base Converter)

这是一个基于 C 语言编写的命令行交互程序。在基础版本之上进行了大幅升级，现已全面支持**十进制（Decimal）**、**二进制（Binary）**、八进制（Octal）**和**十六进制（Hexadecimal）**之间的 12 种相互转换，并且**完美支持小数（浮点数）的转换。

## ✨ 核心特性

* **全能转换**：支持 2、8、10、16 进制之间的任意互转，共计 12 种模式。
* **小数支持**：不仅支持整数，还能精确处理带有小数点的数值转换（默认保留 8 位小数）。
* **容错处理**：优化了输入逻辑，加入清空缓冲区的机制，避免了输入非数字字符时导致的程序死循环。
* **高精度输出**：针对浮点数（IEEE 754）的精度丢失问题，底层加入了微小偏移量（`1e-11`）进行误差补偿，确保输出结果更符合人类直觉。

## ⚙️ 功能菜单

程序运行后，可通过输入以下序号选择对应的功能：

* `1`: 10 to 2 (十进制转二进制)
* `2`: 10 to 8 (十进制转八进制)
* `3`: 10 to 16 (十进制转十六进制)
* `4`: 2 to 10 (二进制转十进制)
* `5`: 2 to 8 (二进制转八进制)
* `6`: 2 to 16 (二进制转十六进制)
* `7`: 8 to 10 (八进制转十进制)
* `8`: 8 to 2 (八进制转二进制)
* `9`: 8 to 16 (八进制转十六进制)
* `10`: 16 to 10 (十六进制转十进制)
* `11`: 16 to 2 (十六进制转二进制)
* `12`: 16 to 8 (十六进制转八进制)
* `0`: 退出程序

## 🚀 编译与运行

请确保你的计算机上安装了 C 语言编译器（如 GCC）。

**1. 编译代码**
打开终端，运行以下命令（假设文件名为 `converter_advanced.c`）。由于代码使用了 `<math.h>` 库，在 Linux 环境下编译时可能需要链接数学库（加上 `-lm` 参数）：

```bash
gcc converter_advanced.c -o converter_advanced -lm

```

**2. 运行程序**

* **Windows:**
```cmd
converter_advanced.exe

```


* **Linux / macOS:**
```bash
./converter_advanced

```

## 🛠️ 技术细节说明

不同于依赖 `printf` 自带占位符的基础版本，本程序实现了底层的数值解析与生成算法：

* **`base_to_double` 函数**：负责将用户输入的字符串按指定的进制位权展开，分别计算整数部分和小数部分，最终转换为标准的 `double` 类型。
* **`print_double_in_base` 函数**：利用 `<math.h>` 中的 `modf` 函数将数值拆分为整数和小数两部分。整数部分使用连续除基取余法输出，小数部分使用连续乘基取整法输出。
